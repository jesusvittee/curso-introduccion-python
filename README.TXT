# Curso de Introducción a Python - Platzi

Este repositorio contiene los apuntes, ejercicios prácticos y proyectos desarrollados durante el **Curso de Introducción a Python** impartido por [Platzi](https://platzi.com).

## 📚 Contenido del Curso

1. Introducción a Python
2. Variables y Tipos de Datos
3. Operadores Aritméticos y Lógicos
4. Estructuras Condicionales (`if`, `else`, `elif`)
5. Ciclos (`for`, `while`)
6. Funciones y Parámetros
7. Listas, Tuplas y Diccionarios
8. Módulos y Paquetes
9. Manejo de Errores (Excepciones)
10. Entrada y Salida de Datos

## 🚀 Requisitos

- Python 3.x instalado
- Editor de texto o IDE (recomendado: [VS Code](https://code.visualstudio.com/))
- Git (opcional, para clonar el repositorio)
